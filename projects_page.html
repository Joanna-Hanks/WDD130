White Water Rafting
In this project, I created a White Water Rafting website to demonstrate my web design and development skills. I learned how to structure pages with HTML, apply consistent and responsive CSS styling, and use color and layout to enhance user experience. This project helped me understand the balance between functionality and visual design in creating an engaging, professional website.
Holiday Challenge
For this project, I styled a pre-existing HTML page with a Halloween theme using only CSS. The assignment helped me practice designing visually engaging pages, applying colors, fonts, and layout creatively, and enhancing a webpage’s appearance without altering its HTML structure.
Country Flags
For this project, I styled country flags using CSS Grid to practice creating different layouts and aligning elements precisely. I gained hands-on experience with grid properties, learning how to control placement, spacing, and alignment while building visually structured designs.
Prophet Card
This assignment focused on developing a prophet profile card using foundational HTML and CSS based on a simple wireframe layout. Through the project, I enhanced my understanding of semantic structure, effective styling techniques, and responsive, accessible web design principles.
Apostle Spotlight
This assignment involved designing a webpage to spotlight the apostles using fundamental HTML and CSS while adhering to a given design layout. Through this project, I developed skills in organizing web content semantically, applying consistent styling, and creating a visually balanced and responsive design that highlights multiple profiles effectively.
Positioning Blocks
This assignment involved matching activity boxes to a reference image by editing only the CSS. It helped me practice using selectors, layout, colors, and typography, while improving my troubleshooting and precision in writing clean, efficient code.
Boarding Pass
This assignment involved recreating a Delta boarding pass from scratch using CSS Grid for layout. Working from a reference image, I learned how to analyze design structure, translate visual elements into organized HTML and CSS, and use grid properties to align content precisely. This project strengthened my skills in layout design, attention to detail, and replicating real-world interfaces with clean, responsive code.
Gallery
I learned how to resize and optimize images using a photo editor, balancing quality and file size. I also practiced using different file formats, organizing project files, and updating image links for proper webpage display.
